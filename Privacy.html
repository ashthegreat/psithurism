(a) Cookies.

A cookie is a small amount of data stored on the hard drive of the individual's computer that allows Customized Fat Loss For Men to identify the individual with his or her corresponding data that resides in Customized Fat Loss For Men's database. You may read more about cookies at http://cookiecentral.com. Individuals who use the Websites need to accept cookies in order to use all of the features and functionality of the Websites.

(b) Web Beacons.

A web beacon is programming code that can be used to display an image on a web page (by using an programming function -- see http://truste.org for more information), but can also be used to transfer an individual's unique user identification (often in the form of a cookie) to a database and associate the individual with previously acquired information about an individual in a database. This allows Customized Fat Loss For Men to track certain web sites an individual visits online. Web beacons are used to determine products or services an individual may be interested in, and to track online behavioral habits for marketing purposes. For example, Customized Fat Loss For Men might place, with the consent of a third party website, a web beacon on the third party's website where fishing products are sold. When Bill, an individual listed in Customized Fat Loss For Men's database, visits the fishing website, Customized Fat Loss For Men receives notice by means of the web beacon that Bill visited the fishing site, and Customized Fat Loss For Men would then update Bill's profile with the information that Bill is interested in fishing. Customized Fat Loss For Men may thereafter present offers of fishing related products and services to Bill. In addition to using web beacons on web pages, Customized Fat Loss For Men also uses web beacons in email messages sent to individuals listed in Customized Fat Loss For Men's database.

(c) New Technology.

The use of technology on the Internet, including cookies and web beacons, is rapidly evolving, as is Customized Fat Loss For Men's use of new and evolving technology.
